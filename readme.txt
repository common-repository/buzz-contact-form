Extremely efficient and easy to use contact us plugin.

== Description ==

This plugin allows you to add contact us form into your wordpress site. It is extremely easy to use and you can edit email used to receive user queries, success message, invalid email entry message, empty field message, invalid captcha entry message, error message color, success message color and background/text color for submit button.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use 'Settings->Buzz Contact Form' screen to configure the plugin.
